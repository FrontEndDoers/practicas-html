Etiquetas Básicas HTML

Este ejemplo muestra como combinar algunas de las etiquetas más básicas de HTML.
Recuerda que es importante entender las diferencias entre ellas

Etiqueta ul+li

Si listamos personas nominadas a los Oscar, dado que el orden no altera el significado, debemos usar ul.

  • Antonio Banderas por Dolor y Gloria
  • Leonardo Di Caprio por Erase una vez en Hollywood
  • Joaquin Phoenix por Joker

Etiqueta ol+li

En el caso de que estemos listando elementos donde el orden es importante, debemos usar ol.

  1. Rusia
  2. Alemania
  3. Brasil
  4. Portugal
